# tinted-aviutl2

[tinted-theming](https://github.com/tinted-theming/home#tinted-theming-all-your-themes-everywhere-) をAviUtl2で使うためのテンプレート。

## 使い方

[`./generated`フォルダ](./generated)にあるAviUtl2用のテンプレートをAviUtl2のデータフォルダ（その他 -> アプリケーションデータ -> アプリケーションデータフォルダ）の`style.conf`にコピーしてください。
